================================================================================
                          HOW IT WORKS
================================================================================

STEP 1: TEAM_MEMBER CREATES REQUEST
  └─ Click "Deploy" button
     Select project → Choose tasks
     Select environment → Add description
     Submit → Status: PENDING
     Notified: Project Manager, Admin

STEP 2: PROJECT_MANAGER REVIEWS
  └─ Goes to Deployments page
     Clicks "Pending Approvals" tab (shows count badge)
     Reviews deployment details
     Clicks "Approve" or "Reject"
     If rejected → Adds reason, Requester notified

STEP 3: PROJECT_MANAGER DEPLOYS (if approved)
  └─ Clicks on approved deployment
     Selects environment (Staging/Testing/Preview)
     Clicks "Deploy Now"
     Status changes to DEPLOYED
     Audit log updated
     Requester notified

STEP 4: VIEW HISTORY ANYTIME
  └─ "History" tab shows all deployments
     Search by description
     Click any to see full timeline
     Activity log shows complete history
